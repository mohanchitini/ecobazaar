EcoBazaar — Eco-Friendly E-Commerce Platform (Work in Progress)

EcoBazaar is a backend-focused e-commerce application built with Spring Boot, aimed at promoting sustainable and eco-conscious shopping. It enables users to browse, add, and purchase eco-friendly products while empowering sellers to list green alternatives.

The project emphasizes clean architecture, scalability, and security, featuring a 3-layer Spring Boot structure (Controller → Service → Repository) and JWT-based authentication for secure user access.

🧩 Technology Stack

Backend: Spring Boot

Security: JWT Authentication

Database: MySQL (Spring Data JPA & Hibernate)

Build Tool: Maven

Language: Java 17

Tools: Postman, STS / IntelliJ IDEA, Lombok

🧱 Current Progress

✅ User registration and login with JWT

✅ Product CRUD operations

✅ Cart management and summary view

🚧 Checkout and Order module under development

🚧 Payment integration planned
